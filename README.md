# install

```sh
git clone https://github.com/alpheratz0/christmas.git
cd christmas
chmod +x ./install.sh
sudo ./install.sh
```

# usage

after install you should be able to run the following commands

```sh
tree.sh
tree.sh -l en
tree.sh -l es
tree.sh -L
```

# how to exit?

just send an interrupt signal to the process pressing ctrl+c
